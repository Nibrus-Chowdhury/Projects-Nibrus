Lesson #4 The Need for Programming Languages

What is the difference between a programming language and natural (every-day) language?

The difference between a programming language and a natural language is that we use a natural language to communicate with each other but we use a programming language to communicate with a computer which makes the language harder to learn. Computers don't understand the conventions of natural languages as they don't have brains but run on very specific commands whereas natural languages do not require or have too much specificity.
